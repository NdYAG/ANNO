读书笔记

豆瓣读书笔记第三方客户端

下载

Markdown 书写

支持 Github Flavour Markdown 语法，即时书写预览，加入代码高亮，图片预览。

跨平台支持

是一款 Chrome App，可以安装在 Windows/Linux/MacOS/ChromeOS 下独立运行。

豆瓣读书笔记与 Evernote

基于豆瓣读书笔记的 OAuth2 API 开发，管理书籍与笔记，支持收藏和导出文件，以及将多篇笔记保存到 Evernote。
